# Utilities

This directory contains utility scripts and tools for ESP8266 development.

## Available Utilities

### upload_files.py
Batch upload multiple files to ESP8266.

**Usage:**
```bash
python upload_files.py /dev/ttyUSB0 file1.py file2.py
```
